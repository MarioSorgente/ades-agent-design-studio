# ADES — Agent Design Studio

ADES is a visual AI-powered design studio for PMs and non-developers.
It helps users design, critique, and evaluate AI agents before implementation.

## Current status

M0 scaffold is in place:

- Next.js + TypeScript + Tailwind
- App Router base structure
- Landing page aligned with ADES product goals

## Planned sequence

1. M0 scaffold ✅
2. M1 route shell (`/`, `/sign-in`, `/dashboard`, `/project/[id]`, `/project/[id]/print`)
3. M2 Firebase wiring
4. M3 Google auth flow
5. M4 dashboard + project CRUD
6. M5 React Flow + Zustand board
7. M6 Firestore persistence
8. M7 generate API
9. M8 critique API
10. M9 export features
11. M10 usage caps
